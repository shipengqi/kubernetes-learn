    Quick Start

    1. 获取源码

    2. hexo-theme-doc-seed 的以下文件拷贝到项目 root 目录下，例如 kubernetes-learn：

    • source 目录
    • _data 目录
    • images 目录
    • package.json
    • _config.yaml
    • .zappr.yaml

    1. 安装依赖

    2. 修改 package.json，直接运行 hexo server 或者 hexo s 可能会找不到命令。

    3. 修改 _config.yml

    4. 启动开发服务，访问 http://localhost:8082 。

    Sidebar

    source 目录下的 _data 目录下的 navigation.yaml 设置 sidebar 和其他的一些配置。

    • logo: navigation Logo
    • main: left sidebar

    对于每个导航项，必须定义一个 type，并根据类型定义 text 和 path 等其他属性。
    每个导航项，也可以定义一个 children，这个属性可以嵌套导航项。

    type

    type 有两种类型：

    • label: 导航项的标签
    • link: 导航项下级文档 link

    link 类型的导航项的 path 的值是文件的路径，但注意扩展名为.html。
